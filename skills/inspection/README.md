# inspection

Inspection skills for APYROBO: patrol routes with scanning and area surveys.

## Skills

| Skill ID | Description | Capability |
|----------|-------------|------------|
| `patrol_route` | Navigate waypoints with obstacle scan at each stop | navigate |
| `scan_area` | Full 360-degree sensor sweep | rotate |

## Installation

```bash
apyrobo pkg install skills/inspection
```

## Dependencies

- `builtin-skills >= 1.0.0`
