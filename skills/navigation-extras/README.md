# navigation-extras

Extended navigation skills for APYROBO: waypoint tours and autonomous docking.

## Skills

| Skill ID | Description | Capability |
|----------|-------------|------------|
| `waypoint_tour` | Navigate a loop of waypoints | navigate |
| `dock_to_charger` | Navigate to dock and connect | navigate |

## Installation

```bash
apyrobo pkg install skills/navigation-extras
```

## Dependencies

- `builtin-skills >= 1.0.0`
